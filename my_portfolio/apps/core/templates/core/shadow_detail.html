Shadow

A comprehensive project showcase and personal identity platform. Built with modern web technologies to be fast, responsive, and aesthetically pleasing.

Django 4.2 Vanilla CSS Glassmorphism Responsive

The Vision

"Shadow" represents my digital identity. It's not just a collection of links, but a crafted experience that reflects my attention to detail and passion for modern web development. The goal was to build a portfolio that stands out through its aesthetics and smooth user interactions.

Unlike generic templates, this project helps me demonstrate my ability to build custom design systems from scratch using vanilla CSS, without relying on heavy frameworks like Bootstrap or Tailwind for the core look.

Key Features

  • ✅ Custom Design System: A unique CSS variable-based system for colors, typography, and spacing.
  • ✅ Glassmorphism UI: Modern, frosted-glass effects using `backdrop-filter` for a premium feel.
  • ✅ Dynamic Animations: Subtle scroll-triggered fade-ins and interactive hover states.
  • ✅ Responsive Grid: A layout that adapts seamlessly from desktop monitors to mobile screens.
  • ✅ Scalable Architecture: Built on Django's app-based structure for easy expansion (e.g., adding Blog or Updates apps).

Under the Hood

Backend

Django 4.2 serves as the robust backbone, handling routing, templating, and data management.

Frontend

HTML5, CSS3 (Variables, Flexbox, Grid), and minimal JavaScript for interactions.

Deployment

WSGI compatible, ready for deployment on platforms like Heroku, AWS, or DigitalOcean.
